# AURAC | Full-Stack Digital & Creative Studio

Aurac is a full-stack digital & creative studio. We help brands look better, work smarter, and grow faster with web, design, marketing, and branding.

## Tech Stack
- **Framework**: React 19 + Vite
- **Styling**: Vanilla CSS
- **Animations**: Framer Motion

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run locally:
   ```bash
   npm run dev
   ```

## Branding
The studio focuses on:
- **Build**: High-performance web applications.
- **Design**: Clean, modern UI/UX.
- **Grow**: Strategic marketing and growth.
- **Create**: Motion graphics and visual storytelling.
